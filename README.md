
# Adder Project — OpenFPGA

## Author
RoyalMaddy08

## Description
A 1-bit Full Adder implemented in Verilog and mapped to an FPGA fabric using OpenFPGA. The Full Adder adds two 1-bit binary numbers with a carry-in and produces a Sum and Carry-out output.

## Files
- `adder.v` — Original Verilog source code
- `adder.blif` — Synthesized netlist (generated by Yosys)
- `adder.act` — Signal activity file
- `task.conf` — OpenFPGA task configuration
- `fabric_bitstream.bit` — Generated FPGA bitstream
- `adder_formal.vcd` — Simulation waveform file
- `vvp_sim_output.txt` — Simulation result (Passed)

## Tools Used
- OpenFPGA (University of Utah)
- Yosys — Logic Synthesis
- ABC — Logic Optimization
- VPR — Place & Route
- Icarus Verilog — Simulation
